# HR Management System

## Overview
This is a web-based HR Management System developed using Laravel. It is designed to streamline employee record management, attendance tracking, and basic HR operations within an organization.

## Features
- Employee record management  
- Attendance tracking system  
- HR data management and reporting  

## Technologies Used
- Laravel  
- PHP  
- MySQL  
- Bootstrap  
- HTML, CSS  

## Installation & Setup
1. Clone the repository  
2. Run `composer install`  
3. Copy `.env.example` to `.env`  
4. Configure database in `.env` file  
5. Generate application key: `php artisan key:generate`  
6. Run migrations: `php artisan migrate`  
7. Start the server: `php artisan serve`

## Purpose
This project was developed to automate HR processes and improve efficiency in managing employee data.
